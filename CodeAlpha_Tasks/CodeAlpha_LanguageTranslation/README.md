# 🌐 Task 1 — Language Translation Tool
**CodeAlpha AI Internship**

## Features
- Translate between 20+ languages using Google Translate (free, no API key)
- Auto-detect source language
- Swap source/target language with one click
- Copy translated text to clipboard
- Text-to-speech for translated output (optional)
- Clean dark-themed GUI built with Tkinter

## Setup & Run
```bash
pip install -r requirements.txt
python app.py
```

## Tech Stack
- **Python** + **Tkinter** (GUI)
- **deep-translator** (free Google Translate wrapper)
- **pyttsx3** (text-to-speech, optional)

Dark-themed UI with side-by-side input/output panels, language dropdowns, and action buttons.
